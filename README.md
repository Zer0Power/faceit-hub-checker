### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Zer0Power/faceit-hub-checker
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Edit index File & Replace ur API_KEY & ur HUB_ID
4. Run the Project
   ```sh
   npm run test
   ```
